Free Code Camp: URL Shortener Microservice

How it Works

  • I can pass a URL as a parameter to the pathname, and I will receive a shortened URL in the JSON response.
  • If I visit the shortened URL mentioned, I will be taken to my original link

Examples

Creation

http://localhost:3500/http://google.com

Creation Output

{ "original_url":"http://google.com", "short_url":"http://localhost:3500/3453" }

Usage

Going to http://localhost:3500/3453 will redirect to http://google.com
